ABSORB

Surface Adsorption Calculation Platform

Welcome to ABSORB

This is a placeholder page for the backend API. The full Vue.js frontend should be built and served separately.

Core Features

  • Automatic surface adsorption site identification (hollow sites and on-top sites)
  • Geometry optimization with rotation algorithms
  • Adsorption energy calculations using CHGNet
  • Real-time calculation log streaming
  • 3D interactive result visualization
  • Calculation history management
  • Result download as ZIP archives

Technology Stack

  • Backend: Flask + Python + ASE + CHGNet
  • Frontend: Vue.js 3 + Vite + Plotly.js
  • Architecture: Modular with factory patterns

API Endpoints

  • POST /run-calculation - Submit new calculation
  • GET /check-status/<session_id> - Check calculation status
  • GET /stream-logs/<session_id> - Stream calculation logs
  • GET /get-viz-data/<session_id>/<filename> - Get visualization data
  • GET /get-results - Get all results
  • GET /download-result/<session_id> - Download results ZIP
  • GET /api/health - Health check
